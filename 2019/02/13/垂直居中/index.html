    如何实现元素居中
    2019/02/13 前端 CSS

    总结一下居中方法。

    水平居中

    行内元素

    利用 text-align 可以实现在块级元素内部的行类元素居中。这个方法对 inline、inline-block、inline-table 和 inline-flex 元素水平居中都有效果

    .parent {
  /* 父容器上设置 */
  text-align: center;
}

    如果块级元素内部包含着一个块级元素，可以将块级元素转化为行内块元素。在使用上述方法就能达到水平居中。

    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    text-align: center;
  }
  .child {
    display: inline-block; /*改变子元素成为行内块元素*/
  }
</style>

    块级元素

    1. margin: 0 auto
    .child {
  width: 100px; /*确保该块级元素定宽*/
  margin: 0 auto;
}
    1. 使用 absolute + transform(如果父元素定宽，可以使用 absolute + 负 margin)
      父元素相对定位，子元素绝对定位。往右移动子元素，移动距离为父容器的一半，然后左移子元素宽度的一半达到居中。
    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    position: relative;
  }
  .child {
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
  }
</style>
    1. 使用 flex + justify-content(flex 布局)
    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    display: flex; /*默认主轴为row*/
    justify-content: cneter;
  }
</style>

    垂直居中

    行内元素

    1. 单行内联元素
      height 值和 line-height 值相等即可
    <div>
  <span>认真学习前端的人</span>
</div>
<style>
  div {
    height: 100px;
    line-height: 100px;
  }
</style>
    1. 多行内联元素
      使用 flex + justify-content + flex-deriction: column(主轴方向为竖直方向)
    <div class="parent">
  <p>
    React 是一个用于构建用户界面的 JAVASCRIPT 库。
    React主要用于构建UI，很多人认为 React 是 MVC 中的 V（视图）。 React 起源于
    Facebook 的内部项目，用来架设 Instagram 的网站，并于 2013 年 5 月开源。
    React 拥有较高的性能，代码逻辑非常简单，越来越多的人已开始关注和使用它。
  </p>
</div>
<style>
  .parent {
    height: 150px;
    display: flex;
    flex-direction: column;
    justify-content: center;
  }
</style>

    块级元素

    1. 使用 absolute + transform(如果父元素定高，可以使用 absolute + 负 margin)
      通过绝对定位元素距离顶部 50%，借助 CSS3 中的 transform 属性向 Y 轴反向偏移 50%的方法实现垂直居中。
    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    position: relative;
  }
  .child {
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
  }
</style>
    1. 使用 flex + align-items
    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    display: flex; /*默认主轴为row*/
    align-items: cneter; /*交叉轴*/
  }
</style>

    水平垂直居中

    1. 使用 absolute + transform
      在未知元素的高宽的情况下实现元素的垂直居中。(已知宽高可以用负 margin)
    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    position: relative;
  }
  .child {
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
  }
</style>
    1. 使用 flex 布局
      在主轴和纵轴都设置对齐方式
    <div class="parent">
  <div class="child">demo</div>
</div>
<style>
  .parent {
    display: flex; /*默认主轴为row*/
    align-items: cneter; /*交叉轴*/
    justify-content: center; /*主轴*/
  }
</style>
