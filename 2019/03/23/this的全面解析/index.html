this是JavaScript中最复杂的机制之一。他很特殊，被自动定义在所有函数的作用域中。

每个函数的this是在调用的时候被绑定的，完全取决于函数的调用位置。

紧接着，走进this的世界中。

调用位置

关于调用位置，调用位置就是函数在代码中被调用的位置(而不是声明位置)。

其实最重要的还是分析调用栈(就是为了达到当前执行位置所调用的所有函数)。
